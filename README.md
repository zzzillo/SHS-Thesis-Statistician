# Statistical Analysis (Senior High School Thesis)

## About

This repository contains the statistical analysis conducted for a Senior High School thesis.

As the statistician for this study, I was responsible for handling the data, performing statistical procedures, and preparing the analysis workflow to support the research.

---

## Scope of Work

The analysis includes:

- Data cleaning and preprocessing  
- Descriptive statistical analysis  
- Normality testing (Shapiro–Wilk)  
- Correlation analysis (Spearman)  
- Likert scale processing and interpretation  
- Outlier detection using IQR method
